# Recruitment Data Analysis Dashboard

## Overview
This project analyzes recruitment data to identify key hiring trends and improve decision-making.

## Objectives
- Track recruitment KPIs
- Analyze submission and selection rates
- Identify bottlenecks in hiring process

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## Key Insights
- Identified roles with highest demand
- Analyzed time-to-fill for different positions
- Improved visibility into recruitment pipeline
